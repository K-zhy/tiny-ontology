# 自然语言测试题库

基于当前 ontology schema 与种子数据生成，覆盖 Student、Teacher、Course、Score 四类对象，以及 Link、Function、ObjectSet、Action 等核心语义能力。

## 使用说明

- 低难度题目主要验证单对象查询、普通属性过滤、派生属性读取。
- 中难度题目主要验证跨对象关系查询、排序、模糊搜索、ObjectSet 与函数调用。
- 高难度题目主要验证多跳推理、多条件组合、聚合比较、写操作后的状态变化。
- 涉及新增、修改、删除成绩或调整授课教师的题目会修改数据库，建议每轮测试前执行 `python seed_data.py` 重置数据。

## 低难度

| 编号 | 自然语言问题 | 覆盖能力 |
|---|---|---|
| L1 | 现在系统里有哪些学生？ | Student 基础查询 |
| L2 | 张三今年多大，在哪个班？ | Student 属性查询 |
| L3 | 有哪些女学生？ | 条件过滤 |
| L4 | 2024-春学期开了哪些课程？ | Course 属性过滤 |
| L5 | 高等数学是多少学分？是哪学期开的？ | Course 详情查询 |
| L6 | 大学英语是谁教的？ | Course -> Teacher 单跳关系 |
| L7 | 信息学院有哪些老师？ | Teacher 条件过滤 |
| L8 | 王五的所有成绩分别是多少？ | Student -> Score 关联查询 |
| L9 | 张三的平均分是多少？ | Student 派生属性 avgScore |
| L10 | Python编程这门课的通过率是多少？ | Course 派生属性 passRate |

## 中难度

| 编号 | 自然语言问题 | 覆盖能力 |
|---|---|---|
| M1 | 名字里带“张”的对象有哪些？包括学生、老师和课程。 | Nameable 接口，跨类型搜索 |
| M2 | 张教授现在教哪些课？这些课总共有多少学分？ | Teacher <- Course 反向关系，聚合 |
| M3 | 高等数学成绩最高的学生是谁，分数是多少？ | Score 排序，跨 Link 查询 |
| M4 | 数据结构这门课前 3 名是谁？ | getTopStudents / 排名 |
| M5 | 当前有哪些优秀学生？ | ObjectSet: TopStudents |
| M6 | 当前有哪些及格课程？ | ObjectSet: PassedCourses |
| M7 | 计算机2201班的学生按平均分从高到低怎么排？ | 条件过滤 + 派生属性排序 |
| M8 | 孙七有哪些课程没及格？ | Student + Score + Course 联合过滤 |
| M9 | 王五修过 Python编程 吗？如果修过，成绩是多少？ | 跨对象存在性判断 |
| M10 | 2024-春学期里，张教授教授了哪些课程？ | 多条件组合过滤 |
| M11 | 哪门课平均分最高？哪门课通过率最低？ | Course 聚合比较 |
| M12 | 把张三的成绩单列出来，并显示每门课对应的授课老师。 | Student -> Score -> Course -> Teacher 多跳查询 |

## 高难度

| 编号 | 自然语言问题 | 覆盖能力 |
|---|---|---|
| H1 | 哪些学生的平均分高于自己所在班级的平均水平？ | 分组聚合，对比推理 |
| H2 | 有没有学生在自己所有已修课程里都及格？把他们按平均分从高到低列出来。 | 全量条件判断，派生属性排序 |
| H3 | 2024-春学期里，由数学学科老师授课且通过率超过 80% 的课程有哪些？ | Course + Teacher 条件组合，派生属性过滤 |
| H4 | 同时上了高等数学和数据结构，而且这两门都超过 80 分的学生有哪些？ | 多课程交集，多条件过滤 |
| H5 | 找出至少修了 3 门课且平均分超过 85 分的学生。 | 计数聚合 + 派生属性过滤 |
| H6 | 先给李四补录一条 Python编程 78 分、考试日期为 2024-12-20 的成绩，再告诉我他新的平均分是多少。 | createScore + 派生属性重算 |
| H7 | 把赵六的高等数学成绩改成 90 分后，他会不会进入优秀学生集合？ | updateScore + ObjectSet 变化验证 |
| H8 | 删除李四的数据结构成绩后，他还剩下几门不及格课程？分别是什么？ | deleteScore + 结果重算 |
| H9 | 把 Python编程 改为由张教授授课后，张教授名下共有几门课，总学分是多少？ | assignTeacher + 关系重算 |
| H10 | 哪位老师名下课程的平均分最高？同时列出这位老师所教每门课的平均分。 | Teacher -> Course -> Score 聚合推理 |
| H11 | 对每个班级，找出平均分最高的学生，并说明他成绩最高的一门课是什么。 | 分组 Top1 + 多跳查询 |
| H12 | 如果搜索关键词“王”，结果里会出现哪些学生、老师和课程？请按对象类型分组展示。 | Nameable 搜索 + 结果分组整理 |

## 额外可选题目

如果你想进一步测试系统对多轮规划和复杂解释的稳定性，还可以追加下面这些问题：

- 帮我比较张三和王五的成绩结构，谁更偏科？请给出依据。
- 哪些课程是计算机2201班学生整体表现更好、但其他班表现一般的？
- 如果把所有低于 60 分的成绩都视为待补考，当前最需要重点关注的学生是谁？为什么？
- 哪位老师的课程覆盖学生最广？请同时列出涉及的学生班级。
- 现在有哪些课程还没有被任何英语2201班学生修读？
