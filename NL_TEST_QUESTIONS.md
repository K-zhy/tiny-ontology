# 自然语言测试题库

基于当前 ontology schema 与种子数据生成，覆盖 Student、Teacher、Course、Score 四类对象，以及 Link、Function、ObjectSet、Action 等核心语义能力。

## 使用说明

- 低难度题目主要验证单对象查询、普通属性过滤、派生属性读取。
- 中难度题目主要验证跨对象关系查询、排序、模糊搜索、ObjectSet 与函数调用。
- 高难度题目主要验证多跳推理、多条件组合、聚合比较、写操作后的状态变化。
- 下表中的“预期结果”均基于当前种子数据。
- 涉及新增、修改、删除成绩或调整授课教师的题目会修改数据库，建议每轮测试前执行 `python seed_data.py` 重置数据。
- 对于写操作题，默认按“每道题单独从初始种子数据开始执行”理解，不串联上一题的数据库状态。

| H3 | 2024-春学期里，由数学学科老师授课且通过率超过 80% 的课程有哪些？ | Course + Teacher 条件组合，派生属性过滤 | 无。张教授教授的高等数学通过率 80.0%，线性代数 75.0%，都未超过 80% |
| H4 | 同时上了高等数学和数据结构，而且这两门都超过 80 分的学生有哪些？ | 多课程交集，多条件过滤 | 张三、王五 |
| H5 | 找出至少修了 3 门课且平均分超过 85 分的学生。 | 计数聚合 + 派生属性过滤 | 张三、王五 |
| H6 | 先给李四补录一条 Python编程 78 分、考试日期为 2024-12-20 的成绩，再告诉我他新的平均分是多少。 | createScore + 派生属性重算 | 新平均分为 63.6 |

| H8 | 删除李四的数据结构成绩后，他还剩下几门不及格课程？分别是什么？ | deleteScore + 结果重算 | 还剩 1 门：线性代数 55 |
| H9 | 把 Python编程 改为由张教授授课后，张教授名下共有几门课，总学分是多少？ | assignTeacher + 关系重算 | 共 3 门课：高等数学、线性代数、Python编程；总学分 10 |
| H10 | 哪位老师名下课程的平均分最高？同时列出这位老师所教每门课的平均分。 | Teacher -> Course -> Score 聚合推理 | 李教授最高，名下只有大学英语，课程平均分 84.0 |
| H11 | 对每个班级，找出平均分最高的学生，并说明他成绩最高的一门课是什么。 | 分组 Top1 + 多跳查询 | 计算机2201：张三，最高分课程高等数学 95；计算机2202：王五，最高分课程大学英语 95；数学2201：赵六，最高分课程线性代数 81；英语2201：孙七，最高分课程大学英语 91 |
| H12 | 如果搜索关键词“王”，结果里会出现哪些学生、老师和课程？请按对象类型分组展示。 | Nameable 搜索 + 结果分组整理 | 学生：王五；老师：王教授；课程：无 |

## 额外可选题目

如果你想进一步测试系统对多轮规划和复杂解释的稳定性，还可以追加下面这些问题。下面同时给出参考结果：

| 编号 | 自然语言问题 | 参考结果 |
|---|---|---|
| X1 | 帮我比较张三和王五的成绩结构，谁更偏科？请给出依据。 | 张三更偏科。张三成绩范围 72 到 95，波动更大，大学英语 72 明显低于其他课程；王五成绩整体更均衡，范围 82 到 95 |
| X2 | 哪些课程是计算机2201班学生整体表现更好、但其他班表现一般的？ | 严格说没有特别典型的一门。若给近似答案，高等数学最接近，因为计算机2201班平均 78.5，高于数学2201和英语2201，但仍低于计算机2202 |
| X3 | 如果把所有低于 60 分的成绩都视为待补考，当前最需要重点关注的学生是谁？为什么？ | 李四。因为他有两门待补考：线性代数 55、数据结构 45，且平均分只有 60.0 |
| X4 | 哪位老师的课程覆盖学生最广？请同时列出涉及的学生班级。 | 张教授和王教授并列，均覆盖 5 名学生；涉及班级为计算机2201、计算机2202、数学2201、英语2201 |
| X5 | 现在有哪些课程还没有被任何英语2201班学生修读？ | 线性代数、数据结构 |
