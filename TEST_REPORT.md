# 自然语言测试报告

基于当前 ontology schema 与种子数据，覆盖 Student、Teacher、Course、Score、TeachingAssignment 五类对象，以及授课关系、Function、ObjectSet、Action 等核心语义能力。

基线数据：10 名学生、7 名教师、9 门课程、12 条授课关系、40 条成绩。每道题独立从初始种子数据开始测试。

通过率：**36/39（92.3%）**
